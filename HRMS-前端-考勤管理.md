# 2. 详细设计

## 2.1 前端迭代目标

1. 考勤规则配置：考勤组 CRUD（含班次时间配置、适用人员选择、节假日管理）
2. 打卡功能：网页端上班/下班打卡按钮，打卡状态实时反馈
3. 考勤日历：日历视图标记每日出勤状态（颜色标识），HR可查看任意员工
4. 请假管理：请假列表 + 请假表单 + 审批进度 + 假期余额展示
5. 补卡/加班：补卡申请表单、加班申请表单
6. 考勤统计：个人/部门统计 + AntV 图表（折线图/饼图/柱状图/日历图）

## 2.2 迭代具体描述

### 2.2.1 考勤规则配置页

##### UI&交互

- 考勤组列表（表格）+ 新增/编辑弹窗
- 适用人员选择器：支持按部门/职位/个人三种维度添加成员
- 班次时间配置：TimePicker 选择上下班时间
- 节假日管理：日历面板点选配置假日

##### 考勤组表单字段

| 字段名称 | 说明 | 输入方式 | 是否必填 | 字段类型 | 数据源 |
| --- | --- | --- | --- | --- | --- |
| 考勤组名称 | 如"标准工时组" | Input | Y | string | - |
| 班次类型 | 固定班/弹性班/排班制 | Radio | Y | number | 枚举 |
| 上班时间 | 如 09:00 | TimePicker | Y | time | - |
| 下班时间 | 如 18:00 | TimePicker | Y | time | - |
| 中午休息 | 如 12:00-13:00 | TimePicker.RangePicker | N | [time,time] | - |
| 弹性最早 | 弹性班显示 | TimePicker | 条件必填 | time | - |
| 弹性最晚 | 弹性班显示 | TimePicker | 条件必填 | time | - |
| 迟到阈值 | 默认15分钟 | InputNumber | Y | number(分钟) | - |
| 早退阈值 | 默认15分钟 | InputNumber | Y | number(分钟) | - |
| 适用人员 | 按部门/职位/个人 | Transfer + TreeSelect | Y | array | 部门树/职位列表/员工搜索 |

##### 工作日设置

- 标准工作日复选框组：周一~周五默认勾选
- 法定节假日：日历面板，可点击标记节假日/调休工作日
- 数据持久化到 holiday 表

### 2.2.2 打卡页面

##### UI&交互

- 居中大卡片布局
- 顶部：当前日期 + 时间（实时刷新）
- 两个打卡按钮：上班打卡（蓝色）、下班打卡（绿色）
- 按钮状态：
  - 今日未打卡 → 按钮可点击
  - 今日已打卡 → 按钮置灰 + 显示打卡时间
  - 非工作日 → 显示"今日休息"
- 打卡结果 Toast 提示（正常/迟到/早退）
- IP/GPS 提示：不在允许范围内时弹窗提示

##### 前端逻辑

```typescript
// 页面加载时获取今日打卡状态
GET /api/v1/attendance/records?month=2024-07  // 找到今日记录
// 点击打卡
POST /api/v1/attendance/clock-in   // 返回打卡时间 + 判定结果
POST /api/v1/attendance/clock-out
// 打卡成功后刷新今日状态
```

### 2.2.3 考勤日历页

##### UI&交互

- 日历视图（FullCalendar / AntD Calendar）
- 每个日期格子标记颜色：
  - 绿色圆点 = 正常出勤
  - 橙色圆点 = 迟到 / 早退
  - 红色圆点 = 旷工 / 缺勤
  - 蓝色圆点 = 请假
  - 灰色圆点 = 缺卡
- 点击某天弹出 Popover：详细打卡时间 + 状态说明
- 顶部月份切换 + 汇总数据卡片（出勤天数/迟到/早退/请假天数）
- HR视角：员工选择器切换查看他人

##### 数据源

GET /api/v1/attendance/records?month=2024-07&employeeId=(可选)

### 2.2.4 请假管理

##### 请假列表页

- 顶部分段：我的假期余额卡片（年假余额 xx天 ｜ 调休余额 xx天）
- Tab 状态筛选：全部 / 草稿 / 审批中 / 已通过 / 已拒绝 / 已取消
- 表格列：请假类型、起止时间、天数、事由摘要、状态（Tag）、操作
- 「申请请假」按钮 → 跳转请假表单

##### 请假表单页

| 字段名称 | 说明 | 输入方式 | 是否必填 | 字段类型 | 数据源 |
| --- | --- | --- | --- | --- | --- |
| 请假类型 | 年假/病假/事假/婚假/产假/丧假/调休 | Select | Y | number | 枚举 |
| 开始时间 | 日期+时段（上午/下午） | DatePicker + Select | Y | datetime | - |
| 结束时间 | 日期+时段 | DatePicker + Select | Y | datetime | - |
| 请假天数 | 系统自动计算 | 只读 | - | number | calc(结束-开始) |
| 请假事由 | 文本 | TextArea | Y | string | - |
| 工作交接人 | 指定交接人 | Select + 搜索 | N | number | 员工搜索 |
| 附件 | 病假/婚假/产假需上传 | Upload | 条件必填 | file[] | - |

特殊交互：
- 年假/调休类型选择时显示剩余余额
- 时间重叠校验（与已有请假不冲突）
- 选病假 > 1天时提示"需上传医院证明"

##### 审批进度

- 请假详情页底部展示审批 Timeline
- 已通过的请假显示绿色，审批中显示蓝色，已拒绝显示红色

##### 所需API

| 接口 | 说明 |
| --- | --- |
| GET /api/v1/leave-balance | 假期余额 |
| POST /api/v1/leave-applications | 提交请假 |
| GET /api/v1/leave-applications | 请假列表 |
| GET /api/v1/leave-applications/{id} | 请假详情（含审批进度） |
| PUT /api/v1/leave-applications/{id}/cancel | 取消申请 |

### 2.2.5 补卡申请

##### 表单字段

| 字段名称 | 说明 | 输入方式 | 是否必填 | 字段类型 |
| --- | --- | --- | --- | --- |
| 补卡日期 | 选择缺卡日期 | DatePicker | Y | date |

### 2.2.6 考勤统计页（AntV 可视化）

##### 4个图表 + 2个汇总表格

| 图表/表格 | 图表类型（AntV） | 数据内容 | 适用角色 |
| --- | --- | --- | --- |
| 部门出勤率趋势 | @antv/g2 折线图 | 近6个月各部门出勤率 | HR/主管 |
| 请假类型分布 | @antv/g2 饼图 | 当月各类请假占比 | HR |
| 迟到早退排行 | @antv/g2 柱状图 | 部门维度对比 | HR/主管 |
| 考勤日历图 | 自定义日历热力图 | 每日出勤状态矩阵 | 员工/HR |
| 个人月度汇总 | 表格 + 统计卡片 | 出勤天/迟到/早退/旷工/请假 | 所有人 |
| 部门月度汇总 | 表格 | 部门出勤率/迟到率/请假率 | HR/主管 |

##### 图表交互

- 饼图：hover 显示请假类型 + 天数 + 占比
- 柱状图：点击部门柱下钻查看该部门人员明细
- 折线图：支持切换月份范围（3/6/12个月）
