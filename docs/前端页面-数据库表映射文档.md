## 二、13 张数据库表速查

| # | 表名 | 模块 | 行数预估 | 说明 |
|---|------|------|----------|------|
| 1 | `user` | 基础 | N | 登录认证 + 5角色权限 |
| 2 | `department` | 组织 | 十~百 | 部门树（自关联 parent_id） |
| 3 | `position` | 组织 | 十~百 | 职位+职级合一字典 |
| 4 | `employee` | 员工 | N | 员工档案（1:1 扩展 user） |
| 5 | `attendance_record` | 考勤 | N×30/月 | 打卡+请假+加班+外出四合一 |
| 6 | `approval_flow` | 审批 | N×数次/月 | 通用审批引擎（6类业务） |
| 7 | `salary_account` | 薪资 | 十 | 薪资账套模板 |
| 8 | `salary_item` | 薪资 | 百 | 账套下的工资项目+公式 |
| 9 | `employee_salary` | 薪资 | N | 员工薪资档案（基数） |
| 10 | `salary_batch` | 薪资 | 12/年 | 月度核算批次 |
| 11 | `salary_detail` | 薪资 | N×12/年 | 工资条明细 |
| 12 | `salary_change_history` | 薪资 | N×数次/年 | 调薪历史审计 |
| 13 | `income_tax_cumulative` | 薪资 | N×12/年 | 个税累计预扣 |

### 页面 11：月度薪资核算批次列表页

| 属性 | 内容 |
|------|------|
| **路由** | `/salary/batch` |
| **可见角色** | `admin`, `hr`, `finance` |
| **页面类型** | 统计卡片 + 分页表格 + 状态驱动按钮 |

#### 涉及数据库表

| 表名 | 操作 | 说明 |
|------|------|------|
| `salary_batch` | **R/W/D** | 列表主数据源；新建(INSERT status=0)、删除(DELETE status=0)、状态流转(UPDATE status) |
| `salary_detail` | **R/W** | 计算时批量 INSERT；调整时 UPDATE；重算时先 DELETE 再 INSERT |
| `employee_salary` | **R** | 计算时读取薪资基数 |
| `salary_item` | **R** | 计算引擎按 item_type 选择 Calculator |
| `employee` | **R** | 过滤在职员工 |
| `attendance_record` | **R** | 考勤扣款计算器读取请假/迟到数据 |
| `income_tax_cumulative` | **R/W** | 累计预扣法：读上月累计 → 算本月 → INSERT/UPDATE |

#### 状态流转与表操作

```
草稿(0) ──[执行计算]──→ 计算中(1) ──[异步完成]──→ 待确认(2)
                          │                         │
                          │ 批量 INSERT salary_detail │ [提交审批]
                          │ INSERT income_tax        │ status→3
                          │ UPDATE salary_batch 汇总   ↓
                                                  审批中(3)
                                                  │       │
                                         [审批通过]│       │[驳回]
                                                  ↓       ↓
                                              已通过(4)  已驳回(6)
                                                  │       │
                                         [标记发放]│       │[重新提交]
                                                  ↓       ↓
                                              已发放(5)  回到待确认(2)
                                              (终态)
```

#### 对应 API

| 方法 | 路径 | 涉及表 |
|------|------|--------|
| GET | `/api/v1/salary-batches` | `salary_batch` |
| POST | `/api/v1/salary-batches` | `salary_batch` |
| POST | `/api/v1/salary-batches/{id}/calculate` | `salary_detail` + `income_tax_cumulative` + `salary_batch` + `employee_salary` + `salary_item` + `attendance_record` |
| GET | `/api/v1/salary-batches/{id}/preview` | `salary_detail` + `salary_batch` + `employee` + `department` |
| GET | `/api/v1/salary-batches/{id}/anomalies` | `salary_detail` (is_abnormal > 0) |
| PUT | `/api/v1/salary-batches/{id}/adjust` | `salary_detail` |
| POST | `/api/v1/salary-batches/{id}/submit` | `salary_batch` |
| POST | `/api/v1/salary-batches/{id}/approve` | `salary_batch` |
| POST | `/api/v1/salary-batches/{id}/reject` | `salary_batch` |
| POST | `/api/v1/salary-batches/{id}/mark-paid` | `salary_batch` |
