## 一、测试概述

### 1.1 测试范围

本测分文档覆盖薪资管理模块的全部后端功能，包括：薪资账套 CRUD + 工资项目管理、员工薪资档案维护 + 调薪历史自动记录、月度薪资核算批次全流程（创建→异步计算→预览→异常检测→调整→审批流转→发放）、工资条自助查询（列表+二次验证+详情+趋势）、薪资统计看板 5 个数据接口。

### 1.2 测试目标

- 验证账套及工资项目的 CRUD 完整性、排序正确性、删除引用保护
- 验证员工薪资档案更新后调薪历史自动记录的准确性和 `change_type` 自动识别
- 验证异步计算流程的正确性（6 种计算器 + 累计预扣法个税 + 异常检测 + 明细写入）
- 验证状态机 7 个状态的流转合法性，非法状态转换被正确拒绝
- 验证幂等性：重跑计算不产生重复数据
- 验证字段级权限（manager 看不到薪资字段、user 只能看自己）
- 验证个税累计预扣法的计算精度
- 验证并发场景下的唯一键约束

### 1.3 测试环境

| 项目 | 说明 |
|------|------|
| 数据库 | MySQL 8.0 |
| 缓存 | Redis 7.x |
| 测试框架 | JUnit 5 + Mockito + Spring Boot Test |
| API 测试工具 | Postman / REST Assured |
| 压力测试 | JMeter |
